# thematermux
Change Thema Termux - Termux Thema Changer

=============================

Allow to set Thema for Termux,Set Colors and Fonts.

Without Install Termux Styling App

=======================--=-==

Install:

git clone https://github.com/kangris-sket/thematermux

cd  thematermux

chmod +x changethema.sh

./changethema.sh

done!
